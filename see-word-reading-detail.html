See Word Reading Teacher App

BACKGROUND

The See Word Reading App project has been running few years. It combines English letters and graphics to help young students study the english phonetic. The See Word Reading Student App works for the students to complete the task that set up by teacher, it will record data, times and clicks. The See Word Reading Teacher App using for check the recored data and set up the task contents, teahcer will analyze the record data to help the students improve them study. The student App works will but the teacher App has an unsatisfactory user experience, we need to improve a better experience for teachers.
